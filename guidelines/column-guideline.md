# 365 Health 콘텐츠 작성 가이드라인

## 1. 핵심 원칙

### 1.1 논문 기반 작성 (Evidence-Based)

**모든 주장은 반드시 학술 논문으로 뒷받침되어야 합니다.**

```markdown
✅ 올바른 예시:
"21년 하버드 연구에 따르면 하루 30분 걷는것 만으로도 심혈관 질환 위험이 35%이나 줄어듭니다."

❌ 잘못된 예시:
"걷기 운동은 심장에 좋습니다."
```

#### 출처 표기 규칙

1. **본문 내 인용**: 핵심 수치나 주장 뒤에 간략히 출처 표시
2. **하단 Sources 섹션**: 클릭 가능한 URL 포함
3. **신뢰도 높은 출처 우선**:
   - 1순위: PubMed, NEJM, JAMA, Lancet, Nature Medicine
   - 2순위: 국립암센터, NIH, WHO, CDC
   - 3순위: 대학 연구소, 학회 발표 자료

#### Frontmatter 출처 형식

```yaml
sources:
  - "저자명 et al. (연도). 논문 제목. 저널명. https://doi.org/..."
  - "기관명. (연도). 보고서 제목. https://..."
```

## 2. 글쓰기 스타일

### 2.1 가독성 포맷팅 규칙

#### 문장 줄바꿈 (필수)

**한 문장은 반드시 한 줄에 작성합니다.** 특히 제목, 부제목, 핵심 요약에서 엄격히 적용합니다.

```markdown
✅ 올바른 예시:
"**공복 커피가 위를 망친다는 것은 과학적으로 입증되지 않은 속설입니다.**
건강한 성인이라면 아침 공복에 커피 한 잔은 문제없습니다.
다만 위장 질환이 있다면 식후로 시간을 바꾸는 것이 현명합니다."

❌ 잘못된 예시:
"**공복 커피가 위를 망친다는 것은 과학적으로 입증되지 않은 속설입니다.** 건강한 성인이라면 아침 공복에 커피 한 잔은 문제없습니다. 다만 위장 질환이 있다면 식후로 시간을 바꾸는 것이 현명합니다."
```

#### 리스트 작성 규칙 (필수)

**3개 이상의 항목을 나열할 때는 반드시 숫자 또는 불릿포인트를 사용합니다.**

```markdown
✅ 올바른 예시:
주의가 필요한 그룹:
1. 역류성 식도염 환자
2. 활동성 위염이나 위궤양 환자
3. 카페인 민감자

❌ 잘못된 예시:
주의가 필요한 그룹은 역류성 식도염 환자, 활동성 위염이나 위궤양 환자, 카페인 민감자입니다.
```

#### 폰트 스타일 규칙

| 스타일 | 사용 여부 | 용도 |
|--------|----------|------|
| **Bold** | ✅ 허용 | 핵심 답변, 중요 수치, 강조 |
| *Italic* | ❌ **금지** | 한글 가독성 저하로 절대 사용 금지 |
| ~~취소선~~ | ⚠️ 제한 | 팩트체크에서 오류 표시 시에만 사용 |

```markdown
✅ 올바른 강조:
"**아니요, 사실이 아닙니다.** 2013년 연구에서..."

❌ 잘못된 강조:
"*아니요, 사실이 아닙니다.* 2013년 연구에서..."
```

### 2.2 두괄식 작성법 (Inverted Pyramid)

**첫 문장에서 핵심 결론을 던지세요. 독자의 이목을 단 1초도 놓치지 마세요.**

```markdown
✅ 두괄식 (Good):
"**암 생존자도 소량의 음주는 가능합니다.** 단, 암 종류에 따라 다릅니다.
대장암 생존자는 주 1잔까지 허용되지만, 두경부암 생존자는 완전 금주가 권고됩니다."

❌ 미괄식 (Bad):
"암은 무서운 병입니다. 많은 사람들이 암 치료 후 건강 관리에 관심을 갖습니다.
그 중에서도 음주에 대한 궁금증이 많은데요... (중략) ...결론적으로 소량은 가능합니다."
```

### 2.3 전문적이지만 쉬운 설명

**의학 용어는 즉시 풀어서 설명하세요.**

```markdown
✅ 올바른 예시:
"인슐린 저항성(몸이 인슐린에 반응하지 않는 상태)이 높아지면..."

❌ 잘못된 예시:
"인슐린 저항성이 증가하면 HOMA-IR 수치가 상승하고..."
```

#### 난이도 기준

| 대상 독자 | 설명 수준 | 예시 |
|----------|----------|------|
| 일반인 | 중학교 2학년 수준 | "혈당이 올라가면" |
| 전문적 내용 | 괄호로 쉬운 설명 추가 | "공복혈당(아침에 측정하는 혈당)" |
| 수치 데이터 | 비율로 직관적 표현 | "10명 중 3명" > "30%" |

## 3. SEO & AEO 최적화

### 3.1 제목 작성 규칙

```markdown
✅ SEO 최적화 제목:
"당뇨병 환자 커피 마셔도 될까? 하루 3잔까지 안전 [연구결과]"

구성 요소:
- 검색 키워드: "당뇨병 환자 커피"
- 궁금증 유발: "마셔도 될까?"
- 구체적 정보: "하루 3잔까지"
- 신뢰도 표시: "[연구결과]"
```

### 3.2 Frontmatter SEO 필드

```yaml
---
title: "검색 노출용 제목 (60자 이내)"
description: "검색 결과에 표시되는 설명 (155자 이내)"
metaTitle: "SEO 최적화된 타이틀 태그 (50-60자)"
metaDescription: "검색 노출용 메타 설명, 핵심 키워드 포함 (150-160자)"
tags:
  - "핵심키워드1"
  - "핵심키워드2"
  - "롱테일키워드"
---
```

### 3.3 AEO (Answer Engine Optimization)

**AI 검색 엔진(ChatGPT, Perplexity, Claude)이 답변으로 인용하도록 최적화**

#### 필수 구조

1. **핵심 답변 박스** - 글 초반에 요약 배치
```markdown
> **핵심 요약**: 당뇨병 환자도 하루 3잔까지 블랙커피는 안전합니다.
> 단, 설탕과 크림은 피하세요. 카페인이 혈당 조절에 도움이 될 수 있습니다.
```

2. **FAQ 섹션** - 자주 묻는 질문 형식
```markdown
## 자주 묻는 질문

### Q: 당뇨병 환자가 커피를 마시면 혈당이 올라가나요?

**A: 블랙커피는 혈당에 거의 영향을 주지 않습니다.** 오히려 연구에 따르면...
```

3. **팩트체크 표** - 핵심 정보를 표로 정리
```markdown
| 주장 | 팩트체크 | 근거 |
|------|---------|------|
| 커피가 혈당을 올린다 | ❌ 거짓 | 블랙커피는 혈당 영향 없음 |
| 설탕 넣으면 위험하다 | ✅ 사실 | 혈당 급상승 유발 |
```

## 4. 콘텐츠 구조 템플릿

### 4.1 팩트체크 글 구조

```markdown
---
title: "[주제] 정말 [주장]일까? [팩트체크]"
category: "wellness"
---

**[첫 문장: 결론을 먼저 제시]**

> **핵심 요약**
> - 포인트 1
> - 포인트 2
> - 포인트 3

## 팩트체크 결과

| 주장 | 판정 | 근거 |
|------|------|------|
| ... | ✅/❌/⚠️ | ... |

## [세부 내용 1]

[논문 기반 설명]

## [세부 내용 2]

[논문 기반 설명]

## 전문가는 이렇게 말합니다

> "[전문가 인용문]"
> — 홍길동, 서울대학교병원 내과 교수

## 실천 가이드

1. **첫 번째 권고사항**
2. **두 번째 권고사항**
3. **세 번째 권고사항**

## 자주 묻는 질문

### Q: [질문 1]?
**A:** [답변]

### Q: [질문 2]?
**A:** [답변]
```

> **⚠️ 중요 1 - 제목 중복 금지**: 본문에서 `# 제목`을 작성하지 마세요!
> 제목은 frontmatter의 `title`에서 자동으로 헤더에 표시됩니다.
> 본문은 바로 첫 문장(결론)부터 시작합니다.

> **⚠️ 중요 2 - 참고문헌은 Frontmatter에만**: 본문에 `## 참고 문헌` 섹션을 작성하지 마세요!
> 출처는 frontmatter의 `sources` 필드에 URL과 함께 작성하면 자동으로 페이지 하단에 클릭 가능한 링크로 표시됩니다.

## 5. 콘텐츠 체크리스트

### 작성 전 확인

- [ ] 검색량 있는 주제인가? (Google Trends, 네이버 데이터랩 확인)
- [ ] 논문/연구 자료가 충분한가?
- [ ] 독자가 진짜 궁금해할 내용인가?

### 작성 중 확인

- [ ] 첫 문장에 결론이 있는가?
- [ ] 모든 주장에 출처가 있는가?
- [ ] 의학 용어를 쉽게 풀어썼는가?
- [ ] 표와 리스트를 적절히 활용했는가?
- [ ] **한 문장 = 한 줄** 규칙을 지켰는가?
- [ ] 3개 이상 나열 시 숫자/불릿포인트를 사용했는가?
- [ ] Italic을 사용하지 않았는가?

### 작성 후 확인

- [ ] 제목이 60자 이내인가?
- [ ] 메타 설명이 155자 이내인가?
- [ ] FAQ 섹션이 포함되어 있는가?
- [ ] Frontmatter `sources`에 URL이 포함되어 있는가?
- [ ] 의료 면책 조항이 포함되어 있는가?

## 6. 금지 사항

### 절대 하지 말아야 할 것

1. **근거 없는 주장**
   - "~라고 합니다", "~라고 알려져 있습니다" ❌
   - "2023년 하버드 연구에 따르면~" ✅

2. **과장/선정적 표현**
   - "이것만 먹으면 암이 낫는다" ❌
   - "암 예방에 도움이 될 수 있다" ✅

3. **의료 행위 권유**
   - "약을 끊으세요" ❌
   - "의사와 상담 후 결정하세요" ✅

4. **이미지 출처 제한**
   - 웹에서 가져온 이미지 사용 ❌ (Unsplash, Pexels 포함)
   - AI 생성 이미지만 허용 ✅ (아래 9번 섹션 참조)

5. **Italic 사용 금지**
   - 한글에서 Italic은 가독성을 심각하게 저하 ❌
   - 강조가 필요하면 **Bold**만 사용 ✅

6. **본문에 참고문헌 섹션 금지**
   - 본문에 `## 참고 문헌` 또는 `## Sources` 작성 ❌
   - Frontmatter `sources`에 URL 포함하여 작성 ✅ (자동으로 링크 표시됨)

## 7. 카테고리 및 태그 시스템

### 7.1 메인 카테고리 (3개)

모든 콘텐츠는 반드시 아래 3개 카테고리 중 하나를 선택합니다.

| 카테고리 | 영문 slug | 설명 |
|---------|----------|------|
| 식단 | diet | 영양, 음식, 다이어트 관련 |
| 질병 | disease | 질환, 치료, 의학 정보 |
| 정신건강 | mental-health | 심리, 정신과, 수면 관련 |
| 암 | cancer | 암 환자 관리, 종양학 |

**중요: 개인 실명이나 병원명을 사용하지 않습니다.**

| 분야 | 검수팀명 |
|------|---------|
| 영양/식이 | 36.5 영양 전문팀 |

## 9. AI 이미지 생성 가이드라인

### 9.1 필수 원칙

**모든 이미지는 AI로 직접 생성해야 합니다. 웹에서 가져온 이미지는 절대 사용 불가.**

| 구분 | 허용 | 금지 |
|------|------|------|
| AI 생성 이미지 | ✅ | - |
| Unsplash/Pexels | ❌ | 라이선스 있어도 사용 금지 |
| 구글 이미지 검색 | ❌ | 저작권 문제 |
| 다른 사이트 이미지 | ❌ | 무단 사용 금지 |

### 9.2 이미지 스타일 규격

**고화질 실사(Photorealistic) 이미지를 생성합니다.**

#### 필수 요구사항

1. **실사 퀄리티**: 사진처럼 보이는 고해상도 이미지
2. **상징적 표현**: 글의 핵심 내용을 시각적으로 전달
3. **의료 친화적**: 건강/의료 콘텐츠에 적합한 깔끔하고 신뢰감 있는 분위기
4. **밝고 긍정적**: 불안감을 주지 않는 따뜻하고 희망적인 톤

### 9.3 프롬프트 가이드

#### 기본 프롬프트 구조

```
[주제 설명], photorealistic, high quality, 8K resolution,
professional healthcare photography style,
warm lighting, clean composition, hopeful atmosphere
```

#### 주제별 프롬프트 예시

**식단/영양 관련**
```
Fresh colorful healthy breakfast with coffee cup and fruits on modern kitchen table,
photorealistic, high quality, natural morning light, warm tones,
professional food photography style, clean and inviting atmosphere
```

**의료/질병 관련**
```
Doctor and patient having friendly consultation in modern bright clinic,
photorealistic, high quality, professional healthcare setting,
warm and reassuring atmosphere, natural lighting, hopeful mood
```

**정신건강 관련**
